# Security Policy

## Reporting Security Issues

If you believe you have found a security vulnerability in Kite, please report it privately by opening a [security advisory](https://github.com/Karnonson/kite/security/advisories/new) on GitHub. **Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.**

We will acknowledge your report and work with you to validate and address the issue.

## Scope

Kite is an open-source toolkit and CLI. Reports about the CLI, scripts, templates, and bundled integrations are in scope.

Kite is inspired by [GitHub Spec Kit](https://github.com/github/spec-kit). Issues that originate in upstream Spec Kit code unrelated to Kite-specific changes should also be reported upstream.
